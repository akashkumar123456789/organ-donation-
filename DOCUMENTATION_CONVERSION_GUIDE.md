# How to Convert PROJECT_DOCUMENTATION.html to Word (.docx)

## Method 1: Using Microsoft Word (Recommended)

1. **Open the HTML file in Microsoft Word:**
   - Right-click on `PROJECT_DOCUMENTATION.html`
   - Select "Open with" → "Microsoft Word"
   - OR open Word first, then File → Open → select the HTML file

2. **The document will open with all formatting preserved:**
   - Cover page
   - Certificates
   - All 11 chapters
   - Complete source code
   - Tables and diagrams
   - Bibliography

3. **Save as Word document:**
   - Click File → Save As
   - Choose location
   - Select "Word Document (*.docx)" from format dropdown
   - Click Save

4. **Optional adjustments in Word:**
   - Adjust margins if needed (Layout → Margins)
   - Add page numbers (Insert → Page Number)
   - Modify fonts if desired
   - Add your actual name, roll number, university details

## Method 2: Using Google Docs

1. **Upload to Google Drive:**
   - Go to drive.google.com
   - Click "New" → "File upload"
   - Select `PROJECT_DOCUMENTATION.html`

2. **Open with Google Docs:**
   - Right-click the uploaded file
   - Select "Open with" → "Google Docs"

3. **Download as Word:**
   - File → Download → Microsoft Word (.docx)

## Method 3: Using LibreOffice Writer (Free)

1. **Open in LibreOffice Writer:**
   - Right-click `PROJECT_DOCUMENTATION.html`
   - Open with LibreOffice Writer

2. **Save as Word:**
   - File → Save As
   - Choose "Microsoft Word 2007-365 (.docx)"
   - Click Save
